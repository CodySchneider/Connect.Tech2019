# DailyContent
This component displays daypart specific details from Daily Forecast data

## Usage

```jsx
import { DailyContent } from 'molecule/DailyContent/DailyContent';

<DailyContent
  highTemp={highTemp}
  lowTemp={lowTemp}
  iconCode={iconCode}
  narrative={narrative}
/>

<DailyContent {...dailyData} />
```
